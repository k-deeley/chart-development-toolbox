SpiderChart Examples

Overview
The SpiderChart manages the display of data measuring several distinct numeric variables recorded from one or more subjects. The chart backdrop is a "web" comprising a series of concentric regular polygons and angular rays.
The chart data comprises a numeric matrix Data of size , where:
  • p is the number of numeric variables (equal to the number of vertices and edges in the polygons),
  • q is the number of subjects (equal to the number of data lines on the chart).
Each row of the chart data represents a variable, and each column represents a complete set of measurements for one subject. Each measured value lies between 0 and 1, which allows distinct variables to be compared easily. To display measured values not lying in the range , the input data matrix should be rescaled on a columnwise basis. For ease of comparison, the chart backdrop has 10 concentric polygons with radii .
We impose the constraint on the number of variables. The lower bound ensures that the chart backdrop has at least 3 nodes, i.e., that the web is at least triangular. The upper bound is purely for visual reasons and ensures that the chart backdrop has at most 20 nodes and does not become too cluttered. There is no restriction on the number of lines q, but in practice the chart becomes tricky to read if q is too large.
There is also an additional chart data property TargetData of size . These values are plotted on the web using a separate line object, which can be toggled on or off as required. Visualizing the TargetData property allows the observed data measurements to be compared against their target values. As above, the target data should lie in the range .

Generate sample chart data for one subject.

Let's start with one subject and assume that there are five measured variables.
rng( "default" )
data = rand( 5, 1 );
Define the node labels.
labels = "Property " + (1:numel( data ));

Create a figure for the chart.

f = exampleFigure( "Name", "SpiderChart Example" );

Create the chart.

SC = SpiderChart( "Parent", f, ...
"Data", data, ...
"LabelText", labels );

Annotate the chart.

The Spider chart has the title and legend methods for annotation.
title( SC, "Spider Chart", "FontSize", 14 )

Add target data and customize the appearance of the target line.

We can add a target data vector to the chart.
SC.TargetData = 0.35 * ones( SC.NumNodes, 1 );
Customize the appearance of the target line.
set( SC, "TargetVisible", "on", "TargetLineWidth", 2 )

Hide the target line.

SC.TargetVisible = "off";

Decrease the number of observed variables.

SC.Data = [0.1; 0.3; 0.5; 0.7];

Increase the number of observed variables.

rng( "default" )
SC.Data = rand( 8, 1 );

Update the labels.

SC.LabelText = ["A", "B", "C", "D", "E", "F", "G", "H"];

Increase the number of subjects.

The number of subjects is defined by the number of columns in the chart data matrix.
SC.Data = rand( 8, 4 );

Decrease the number of observed variables.

SC.Data = SC.Data(1:5, :);

Add a legend.

legend( SC, "Subject " + (1:SC.NumLines) )

Update the target data.

set( SC, "TargetData", 0.55 * ones( SC.NumNodes, 1 ), ...
"TargetVisible", "on" )

Customize the visual appearance of the chart.

set( SC, "LineColors", hsv( SC.NumLines ), ...
"LineWidth", 3, ...
"LabelFontWeight", "bold", ...
"LabelFontSize", 12, ...
"LabelFontAngle", "italic" )
