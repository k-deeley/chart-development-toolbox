ValueAtRisk

Overview
The ValueAtRisk chart displays the distribution of a given return series using a histogram. The chart also displays a non-parametric kernel-smoothed probability density fit (computed using fitdist) and two vertical lines indicating the empirical value at risk and conditional value at risk.
  • The x% value at risk is the th percentile of the return series data. For example, the value at risk can be computed by taking the 5th percentile of the return series values r . In MATLAB code, VaR = prctile( r, 5 ).
  • The x% conditional value at risk is the mean of the returns which are less than the x value at risk. For example, the conditional value at risk can be computed as: CVaR = mean( r( r < VaR ) ), where VaR is the value at risk.
The chart data comprises a numeric vector (Data). The value at risk threshold is set using the VaRLevel property of the chart (the default value is 0.95, corresponding to a threshold).

Create sample return series data from a t-distribution for the chart.

rng( "default" )
d = 0.02 * trnd(10, 2000, 1 );

Create a figure for the chart.

f = exampleFigure( "Name", "ValueAtRisk Example" );

Create the chart.

VR = chart.ValueAtRisk( "Parent", f, ...
"Units", "normalized", ...
"Position", [0, 0, 1, 1], ...
"Data", d );

Change the bar color and transparency.

VR.FaceColor = [0, 0.5, 1];
VR.FaceAlpha = 0.5;

Adjust the value at risk level.

VR.VaRLevel = 0.99;

Update the chart data and value at risk level.

newData = 0.20 * randn( 1000, 1 );
set( VR, "Data", newData, "VaRLevel", 0.95 )
