ImpliedVolatility

Overview
The ImpliedVolatility chart creates a smooth, interpolated implied volatility surface for the observed option data , where
  • T is the expiry time of the option, measured in years;
  • K is the strike price of the option, in currency units (e.g., USD or GBP);
  • Ïƒ is the implied volatility, recorded as a decimal value in [0, 1];
  • S is the underlying asset price, in currency units.
The chart data comprises a table (OptionData) containing four numeric vector variables, namely T, K, Ïƒ and S in that order. The observed option data is overlaid on the implied volatility surface for reference. The interpolation method for each volatility curve, or smile, is one of linear, spline, pchip (the default), makima, Hagan2002 and Obloj2008. The first four methods are non-parametric, and implemented using the interp1 function. The latter two are variants of the SABR model for computing the implied Black volatility, and are implemented using the blackvolbysabr function. In this case the SABR model is calibrated using nonlinear least squares (lsqnonlin).

Load the observed market data.

The market data comprises option expiry times, strike prices and implied volatilities. The times to expiry are divided into different bins, indicated by the variable Group. Volatility smiles corresponding to the times in each bin have similar shapes.
load( fullfile( catalogRoot(), "data", "Option.mat" ), "D" )

Create a figure for the chart.

f = exampleFigure( "Name", "Implied Volatility Example" );

Create the chart.

First, extract the option data for a specific group.
D1 = D(D.Group == "G1", 1:4);
IV = chart.ImpliedVolatility( "Parent", f, ...
"Units", "normalized", ...
"Position", [0, 0, 1, 1], ...
"OptionData", D1 );

Annotate the chart.

xlabel( IV, "Time to expiry (years)", "FontSize", 14 )
ylabel( IV, "Strike price ($)", "FontSize", 14 )
zlabel( IV, "Volatility", "FontSize", 14 )
title( IV, "ImpliedVolatility Chart", "FontSize", 16 )
Add a colorbar.
colorbar( IV )

Create a custom colormap and apply it to the chart.

colormap( IV, money() )

Restrict the strike prices to lie between $60 and $100.

newData = D1(D1.StrikePrice >= 60 & D1.StrikePrice <= 100, :);
IV.OptionData = newData;

Remove the first expiry time from the surface.

newData = D1(D1.ExpiryTime > D1.ExpiryTime(1), :);
IV.OptionData = newData;

Create multiple charts on separate figures.

impVolCharts = [impVolChart( D, "G1" ), ...
impVolChart( D, "G2" ), ...
impVolChart( D, "G3" ), ...
impVolChart( D, "G4" )];

Change the surface interpolation method for a specific chart.

impVolCharts(2).InterpolationMethod = "makima";

Customize the appearance of a specific chart.

set( impVolCharts(3), "Marker", "s", ...
"MarkerSize", 4, ...
"MarkerFaceColor", "m", ...
"MarkerEdgeColor", "k", ...
"Controls", "on" )

Local functions.

Define a custom colormap for use with the implied volatility chart.

function cmap = money()
t = linspace( 0, 1, 256 ).';
cmap = [t, flip( t ), zeros( size( t ) )];
end % money

Create an implied volatility chart for a given group.

function IVC = impVolChart( D, groupID )
% Extract the option data for the given group.
currentGroupOptionData = D(D.Group == groupID, 1:4);
% Create a new figure.
f = exampleFigure( ...
"Name", "Implied Volatility, Group " + groupID );
% Create the chart.
IVC = chart.ImpliedVolatility( "Parent", f, ...
"Units", "normalized", ...
"Position", [0, 0, 1, 1], ...
"OptionData", currentGroupOptionData, ...
"Controls", "off" );
% Annotate and customize the chart.
xlabel( IVC, "Time to expiry (years)", "FontSize", 14 )
ylabel( IVC, "Strike price ($)", "FontSize", 14 )
zlabel( IVC, "Volatility", "FontSize", 14 )
title( IVC, "ImpliedVolatility Chart, Group " + groupID, "FontSize", 16 )
colormap( IVC, money() )
end % impVolChart
